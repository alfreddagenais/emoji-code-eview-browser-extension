Emoji Code Review (https://github.com/alfreddagenais/emoji-code-review-browser-extension) is a project to standardize emojis in git messages when you review the code 🎉 This extension should provide an easy way to search and copy codes or emojis during the code review. It is meant for people who knows which emoji should be used 😈 on code review process 🧩 Features
  • 🔍 Search for a specific emoji code's code/emoji by name or description
  • 🌔 Dark/light mode (⚙️)
  • ❌ Close extension's popup after a emoji code is saved in the clipboard (⚙️)
Features with the (⚙️) can be modified in the settings page. ✨ Usage (with keyboard shortcuts)
  1. search emoji names or descriptions
  2. use tab and Shift+tab to navigate between the shortcodes and emojis
  3. press Enter to copy
Storage The storage is used to keep the settings preferences (see 🧩 Features > Through the settings panel).
